⚙️ Konfigurasi Google Sheets

Masukkan URL Google Apps Script untuk menyimpan data pendaftaran.

1. Buka script.google.com
2. Buat project baru, paste kode di atas
3. Ubah nama sheet menjadi "Pendaftaran" di Google Sheets
4. Deploy sebagai Web App (Execute as: Me, Who has access: Anyone)
5. Copy URL dan paste di field di atas
